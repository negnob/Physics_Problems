## 6. Function Analysis

Consider the function

$$f(x)=3x^2-12x+7$$

We need to identify any local maxima or minima.

---

## Useful Definitions and Formulas

### 1. Local maximum and local minimum
A local maximum is a point where the function has a highest value compared to nearby points.

A local minimum is a point where the function has a lowest value compared to nearby points.

For a quadratic function, the local maximum or minimum is found at the vertex of the parabola.

---

### 2. General form of a quadratic function
A quadratic function has the form

$$f(x)=ax^2+bx+c$$

For such a function, the $x$-coordinate of the vertex is

$$x=-\frac{b}{2a}$$

---

### 3. How to tell if it is a maximum or minimum
- If $a>0$, the parabola opens upward, so the vertex is a minimum
- If $a<0$, the parabola opens downward, so the vertex is a maximum

---

## Step-by-Step Solution

We are given

$$f(x)=3x^2-12x+7$$

So:
- $a=3$
- $b=-12$
- $c=7$

---

### Step 1. Find the $x$-coordinate of the vertex

Use the formula

$$x=-\frac{b}{2a}$$

Substitute the values:

$$x=-\frac{-12}{2\cdot3}$$

$$x=\frac{12}{6}$$

$$x=2$$

So the vertex occurs at

$$x=2$$

---

### Step 2. Find the corresponding $y$-value

Now substitute $x=2$ into the function:

$$f(2)=3(2)^2-12(2)+7$$

$$f(2)=3\cdot4-24+7$$

$$f(2)=12-24+7$$

$$f(2)=-5$$

So the vertex is at

$$\left(2,-5\right)$$

---

### Step 3. Determine whether it is a maximum or minimum

The coefficient of $x^2$ is

$$a=3$$

Since

$$3>0$$

the parabola opens upward.

That means the vertex is a local minimum.

There is no local maximum.

---

## Final Answer

The function has a local minimum at

$$\boxed{(2,-5)}$$

and it has no local maximum.
