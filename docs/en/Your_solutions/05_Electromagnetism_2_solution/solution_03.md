# 3. Biot-Savart Law

## Useful definitions and formulas

For a small current segment:

$$B=\frac{\mu_0}{4\pi}\frac{I\Delta l\sin\theta}{r^2}$$

where:

$I$ — current  
$\Delta l$ — length of wire segment  
$r$ — distance to point $P$  
$\theta$ — angle between current segment and line to point $P$  

Since the segment is perpendicular:

$$\theta=90^\circ$$

$$\sin90^\circ=1$$

## Step-by-step solution

Given:

$$I=3\,A$$

$$\Delta l=0.1\,m$$

$$r=0.2\,m$$

Also:

$$\frac{\mu_0}{4\pi}=10^{-7}$$

Now substitute:

$$B=10^{-7}\frac{3\cdot0.1\cdot1}{(0.2)^2}$$

$$B=10^{-7}\frac{0.3}{0.04}$$

$$B=7.5\times10^{-7}\,T$$

## Final answer

$$B=7.5\times10^{-7}\,T$$
