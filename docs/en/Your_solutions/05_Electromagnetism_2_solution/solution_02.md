# 2. Ampere's Law

## Useful definitions and formulas

Magnetic field around a long straight wire:

$$B=\frac{\mu_0 I}{2\pi r}$$

where:

$\mu_0=4\pi\times10^{-7}\,\frac{T\cdot m}{A}$  
$I$ — current  
$r$ — distance from the wire  

## Step-by-step solution

Distance between wires:

$$10\,cm=0.10\,m$$

The point is midway, so distance from each wire:

$$r=0.05\,m$$

Magnetic field from one wire:

$$B_1=\frac{4\pi\times10^{-7}\cdot5}{2\pi\cdot0.05}$$

$$B_1=2.0\times10^{-5}\,T$$

The currents are in opposite directions, so at the midpoint the magnetic fields have the same direction and add together.

$$B_{total}=B_1+B_2$$

$$B_{total}=2\cdot2.0\times10^{-5}$$

$$B_{total}=4.0\times10^{-5}\,T$$

## Final answer

$$B=4.0\times10^{-5}\,T$$

Direction: perpendicular to the plane of the wires, found by the right-hand rule.
