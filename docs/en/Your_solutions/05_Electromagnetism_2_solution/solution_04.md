# 4. Magnetic Torque

## Useful definitions and formulas

Magnetic torque on a current loop:

$$\tau=NIAB\sin\theta$$

where:

$N$ — number of turns  
$I$ — current  
$A$ — area of loop  
$B$ — magnetic field  
$\theta$ — angle between the magnetic field and the normal to the loop  

The magnetic field is parallel to the plane of the loop, so it is perpendicular to the normal:

$$\theta=90^\circ$$

$$\sin90^\circ=1$$

## Step-by-step solution

Given:

$$I=2\,A$$

$$B=0.3\,T$$

$$N=1$$

Convert dimensions:

$$10\,cm=0.10\,m$$

$$5\,cm=0.05\,m$$

Area:

$$A=0.10\cdot0.05=0.005\,m^2$$

Now calculate torque:

$$\tau=1\cdot2\cdot0.005\cdot0.3\cdot1$$

$$\tau=0.003\,N\cdot m$$

## Final answer

$$\tau=3.0\times10^{-3}\,N\cdot m$$
