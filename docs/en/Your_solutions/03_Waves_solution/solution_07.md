## 7. Standing Wave Modes

### Useful definitions and formulas

For a string fixed at both ends, the standing wave condition is:

$$L=n\frac{\lambda}{2}$$

where:

- $L$ is the string length
- $\lambda$ is the wavelength
- $n$ is the harmonic number

Important idea:

- each antinode corresponds to one half-wavelength section
- so if there are four antinodes, then $n=4$

---

### Step-by-step solution

We are given:

- $L=80\text{ cm}=0.80\text{ m}$
- number of antinodes = $4$

So:

$$n=4$$

Use:

$$L=n\frac{\lambda}{2}$$

Substitute:

$$0.80=4\cdot\frac{\lambda}{2}$$

Simplify the right side:

$$4\cdot\frac{\lambda}{2}=2\lambda$$

So:

$$0.80=2\lambda$$

Now divide both sides by $2$:

$$\lambda=\frac{0.80}{2}$$

$$\lambda=0.40\text{ m}$$

In centimeters:

$$\lambda=40\text{ cm}$$

---

### Final answer

$$\lambda=0.40\text{ m}$$

or

$$\lambda=40\text{ cm}$$
