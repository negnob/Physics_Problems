## 2. String Harmonics

### Useful definitions and formulas

For a string fixed at both ends, the fundamental mode means:
- there is **one antinode**
- the string contains **half of a wavelength**

So the formula is:

$$L=\frac{\lambda}{2}$$

where:

- $L$ is the string length
- $\lambda$ is the wavelength

If we want wavelength, we rearrange it:

$$\lambda=2L$$

Then we use the wave formula:

$$v=f\lambda$$

where:

- $v$ is wave speed
- $f$ is frequency
- $\lambda$ is wavelength

---

### Step-by-step solution

We are given:

- $L=64\text{ cm}=0.64\text{ m}$
- $f=330\text{ Hz}$

---

### 1) Find the wavelength

For the fundamental frequency:

$$\lambda=2L$$

Substitute:

$$\lambda=2\cdot0.64$$

$$\lambda=1.28\text{ m}$$

Explanation:

- the fundamental mode has only half a wavelength on the string
- so the full wavelength is twice the string length

---

### 2) Find the wave speed

Now use:

$$v=f\lambda$$

Substitute:

$$v=330\cdot1.28$$

$$v=422.4\text{ m/s}$$

Explanation:

- frequency tells us how many waves pass each second
- wavelength tells us the size of one wave
- multiplying them gives the speed

---

### Final answer

$$v=422.4\text{ m/s}$$
